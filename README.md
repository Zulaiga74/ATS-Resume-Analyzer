# AI Resume Analyzer

A Streamlit application that analyzes resumes and calculates ATS scores.

## Features

- PDF Resume Upload
- Resume Text Extraction
- Skill Detection
- ATS Score Calculation
- Skills Found and Missing Skills
- Interactive Pie Chart
- Recommendations
- Downloadable Report

## Tech Stack

- Python
- Streamlit
- PyPDF2
- Plotly

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
